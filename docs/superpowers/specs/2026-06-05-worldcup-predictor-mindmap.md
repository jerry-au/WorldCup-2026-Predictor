# 世界杯对战预测 App — 思维导图

```mermaid
mindmap
  root((世界杯预测App))
    平台与架构
      Flutter Android/iOS
      后端驱动计算
        FastAPI / Go Gin
        PostgreSQL + Redis
        Celery 任务队列
        Docker 部署
      通信方式
        REST API
        HTTP 轮询长任务
    混合泊松-Elo概率模型
      Stage1 Elo基础
        综合评分 = 0.6×Elo + 0.3×FIFA + 0.1×身价
        Elo差异 → 期望进球 λ
      Stage2 上下文调整
        近期状态漂移
        大赛经验系数
        教练系数
      Stage3 Dixon-Coles双变量泊松
        平局关联参数 ρ
        P(胜)/P(平)/P(负)
      Stage4 实时修正
        核心伤病
        赔率校正
    投注推荐引擎
      隐含概率转换
        去博彩margin
      价值判断
        EV = P_sys × odds - 1
        ★~★★★推荐等级
      赔率偏差检测
        分歧度 Δ ≥ 15%
        系统置信度 ≥ 60%
        跨平台对比(≥3家)
    功能模块
      对战预测 Tab
        选球队+赛制
        预测展示(百分比/雷达图/维度分解)
        投注建议(赔率对比+EV评分)
        收藏组合
      赛事模拟 Tab
        48队完整赛制图
        蒙特卡洛10000次
        实时进度+概率排行榜
      球队详情 Tab
        48队列表(搜索/筛选)
        详情:基础信息/球员卡片/历史/战绩
      我的 Tab
        收藏/历史/数据状态/关于
      投注推荐 Tab
        今日价值场次(按EV排序)
        强烈推荐★★★
        特别关注(偏差检测)
        投注历史追踪
    后端服务
      数据服务
        球队球员CRUD
        外部API聚合器
        赔率数据管理(多博彩公司)
        Redis缓存
      概率计算引擎
        混合泊松-Elo模型
        蒙特卡洛模拟+异步队列
      投注推荐引擎
        赔率对比分析+EV评分
        偏差检测预警+跨平台对比
      用户模块
        收藏/预测历史/投注记录
    API接口
      teams: 列表/详情
      predict: 单场(含推荐)/模拟/任务查询
      recommendations: 价值投注/偏差预警/单场对比
      odds: 最新赔率/博彩公司列表
      data: 刷新
    数据源
      FIFA排名(月)/Elo评分(赛后)/身价(季)
      俱乐部数据(周)/赔率(每5分钟)/伤病(日)
      降级策略: 拉取失败用上次缓存
    异常处理
      网络不可用/超时重试
      数据不完整降权
      赔率数据异常屏蔽
      模拟任务失败重试
      错误码1001-5000
    开发阶段
      P0 核心闭环+基础投注建议
      P1 完整模拟+偏差检测
      P2 数据实时化+完整推荐体验
      P3 体验打磨+投注历史
```

## 纯文本版（备用）

```
世界杯预测App
├── 平台与架构
│   ├── Flutter（Android + iOS）
│   ├── 后端驱动计算（FastAPI + PostgreSQL + Redis + Celery + Docker）
│   └── 通信：REST API + HTTP 轮询
│
├── 混合泊松-Elo 概率模型
│   ├── Stage1 Elo基础：Elo差异 → 期望进球 λ
│   ├── Stage2 上下文调整：状态/经验/教练乘性修正
│   ├── Stage3 Dixon-Coles双变量泊松：含平局关联ρ → W/D/L概率
│   └── Stage4 实时修正：伤病/赔率校正
│
├── 投注推荐引擎
│   ├── 隐含概率转换（去博彩margin）
│   ├── 价值判断（EV = P_sys × odds - 1 → ★~★★★）
│   └── 赔率偏差检测（Δ≥15% + 置信度≥60% + 跨≥3家平台对比）
│
├── 功能模块
│   ├── Tab1 对战预测：预测 + 投注建议 + 收藏
│   ├── Tab2 赛事模拟：蒙特卡洛 + 晋级概率
│   ├── Tab3 球队详情：信息/球员/历史
│   ├── Tab4 我的：收藏/历史/设置
│   └── Tab5 投注推荐：价值场次/特别关注/历史追踪
│
├── 后端服务
│   ├── 数据服务：CRUD + 外部聚合 + 赔率管理 + 缓存
│   ├── 概率引擎：泊松-Elo + 蒙特卡洛 + 异步
│   ├── 推荐引擎：对比/评分/检测/跨平台
│   └── 用户模块：收藏/预测历史/投注记录
│
├── API接口
│   ├── teams：列表/详情
│   ├── predict：单场(含推荐)/模拟/任务
│   ├── recommendations：价值投注/偏差预警/单场对比
│   ├── odds：赔率/博彩公司
│   └── data：刷新
│
├── 数据源（定时同步）
│   ├── FIFA排名(月) / Elo(赛后) / 身价(季)
│   ├── 俱乐部数据(周) / 赔率(每5分钟) / 伤病(日)
│   └── 降级策略：拉取失败用上次缓存
│
├── 异常处理
│   ├── 网络不可用 → 离线提示 + 缓存
│   ├── 超时 → 单场5s / 模拟60s → 重试
│   ├── 数据不完整 → 降权 + 标注
│   ├── 赔率异常 → 自动屏蔽 + 标注
│   └── 错误码体系：1001-5000
│
└── 开发阶段
    ├── P0：核心闭环 + 基础投注建议
    ├── P1：完整模拟 + 偏差检测
    ├── P2：数据实时化 + 完整推荐体验
    └── P3：体验打磨 + 投注历史
```
